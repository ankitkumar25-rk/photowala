1. PROJECT SUMMARY
------------------
Photowala Gift is a high-end e-commerce platform specializing in premium gifts, 
trophies, mementos, and professional custom printing/industrial services. 
The platform is architected for scalability, security, and a premium "luxury" 
user experience.

The ecosystem is divided into three core modules:
- Backend: Express/Node.js REST API with Prisma ORM and PostgreSQL.
- Frontend Store: Customer-facing React SPA (Vite-powered).
- Frontend Admin: Administrative dashboard for business analytics and order management.

2. TECH STACK & ECOSYSTEM
-------------------------
- Core: Node.js (v22.12.0), Express, React 18, Vite.
- Styling: TailwindCSS + Vanilla CSS with "Luxury Minimalist" design tokens.
- State Management: 
    - Frontend: Zustand (Atomic stores for Auth, Cart, UI state).
    - Data Fetching: TanStack Query (React Query v5) for caching.
- Security: 
    - Authentication: PASETO (Token-based) with httpOnly cookie rotation.
    - Protection: Double-submit CSRF tokens, CORS origin filtering.
- Geospatial & Address:
    - Frontend: Google Maps Places API (Autocomplete).
    - Backend: Google Address Validation API (Verification & Correction).
    - Utilities: Postal Pincode API for automated City/State resolution.
- Infrastructure: 
    - Image Hosting: Cloudinary (Auto-optimization).
    - Email: Resend SDK (Custom branded templates).
    - Database: PostgreSQL (Prisma ORM).

3. DATABASE ARCHITECTURE (PRISMA)
---------------------------------
- Polymorphic Relationship Pattern:
    The 'Payment' model supports multiple order types (Physical 'Order' vs. 'ServiceOrder') 
    using 'internalOrderId' and 'orderType'.
- Role-Based Access Control (RBAC):
    Users are categorized into [CUSTOMER, ADMIN, SUPER_ADMIN].
- Printing Service Schema:
    ServiceOrder model handles complex customization metadata for printing-specific workflows.

4. DESIGN SYSTEM & UI STANDARDS
-------------------------------
- Aesthetic Identity: 
    "Luxury Minimalist" — Using Brand Primary (#5a3f2f) and Secondary (#b88a2f).
    Features custom glassmorphism, backdrop blurs, and spring-based animations.
- Responsive Design:
    - Storefront: Standardized for small viewports; full-width search overlays.
    - Admin Panel: Mobile-first sidebar and stacked dashboard layout.

5. LOGISTICS & CHECKOUT INTELLIGENCE
-------------------------------------
- Google Autocomplete:
    Integrated into Checkout and Profile address modals to reduce input friction.
- Address Validation Flow:
    Non-blocking server-side validation that provides correction suggestions to users 
    to prevent shipping errors.
