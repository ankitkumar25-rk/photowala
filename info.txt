DATABASE & CACHING:
  - PostgreSQL: Primary relational database for all application data.
  - Redis: Distributed cache for Session Management, Rate Limiting, and Order Throttling.
  - Prisma ORM: Modern database toolkit used for schema management and type-safe queries.

BACKEND (REST API):
  - Node.js (v18+) & Express.js (v5+): Scalable server-side architecture.
  - Authentication: 
      * Stateless PASETO (Platform-Agnostic Security Tokens) - Secure JWT alternative.
      * Passport.js: Multi-strategy authentication (Local & Google OAuth 2.0).
  - Security: 
      * Helmet.js: XSS, Clickjacking, and CSP protection.
      * Redis Rate Limiting: Protection against Brute Force and DDoS.
      * Zod: Runtime schema validation for all API inputs.
  - File Handling: Multer for multi-part form data and buffered uploads.

FRONTEND (SPA - Storefront & Admin):
  - React 19: High-performance UI library.
  - Vite: Next-generation frontend build tool and dev server.
  - Styling: Tailwind CSS 4 (Custom brand theme: Deep browns, gold accents).
  - Routing: React Router Dom 7 (Modern data-driven routing).
  - State Management: 
      * Zustand: Minimalist client-side state management.
      * TanStack React Query: Powerful server-state caching and synchronization.
  - UI Components: Lucide React (Iconography), React Hot Toast (Notifications), Recharts (Admin Analytics).

THIRD-PARTY ECOSYSTEM:
  - Razorpay: Payment gateway integration (Payments, Webhooks, Refunds).
  - Cloudinary: Professional-grade media hosting and dynamic image optimization.
  - Resend: Transactional email delivery (Receipts, Password Reset, Verification).
  - Google Cloud Platform: Social login and OAuth integration.
